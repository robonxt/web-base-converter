About Base Converter

Base Converter

Effortlessly convert numerical values between binary, octal, decimal, and hexadecimal bases. Enter a value, select bases, and get instant results.

  • Set bases: Choose From and To (2, 8, 10, 16). Use Swap to flip.
  • Enter value: Only digits valid for the From base are accepted.
  • Features: Live result, one‑click copy, method steps, light/dark, offline.
  • Validation: Invalid characters are flagged.
  • Result: Exact conversion plus method.
  • Digits: Bin 0–1 · Oct 0–7 · Dec 0–9 · Hex 0–9, A–F.

Conversion Methods

Binary (2) → Octal (8)

Group bits in 3s from the right and convert each group to octal.

Example: 101011 → 101 011 → 53

Binary (2) → Decimal (10)

Sum bits × 2^position (from right, starting at 0).

Example: 1011 = 1×2^3 + 0×2^2 + 1×2^1 + 1×2^0 = 11

Binary (2) → Hex (16)

Group bits in 4s from the right and convert each group to hex.

Example: 101111 → 0010 1111 → 2F

Octal (8) → Binary (2)

Map each octal digit to its 3‑bit binary.

Example: 27 → 010 111 → 010111

Octal (8) → Decimal (10)

Sum digits × 8^position.

Example: 37 = 3×8^1 + 7×8^0 = 31

Octal (8) → Hex (16)

Convert to binary, group in 4s, then map to hex.

Example: 27 → 010111 → 0010 1111 → 2F

Decimal (10) → Binary (2)

Divide by 2; read remainders in reverse.

Example: 13 → remainders 1,0,1,1 → 1101

Decimal (10) → Octal (8)

Divide by 8; read remainders in reverse.

Example: 59 → remainders 3,7 → 73

Decimal (10) → Hex (16)

Divide by 16; map 10–15 to A–F; read remainders in reverse.

Example: 43 → remainders B,2 → 2B

Hex (16) → Binary (2)

Map each hex digit to its 4‑bit binary.

Example: 2F → 0010 1111 → 00101111

Hex (16) → Octal (8)

Convert to binary, group in 3s, then map to octal.

Example: 2F → 0010 1111 → 000 101 111 → 057

Hex (16) → Decimal (10)

Sum digits × 16^position.

Example: 2F = 2×16^1 + 15×16^0 = 47
